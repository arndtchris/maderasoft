--- Couche Model ---

La couche model contient des « Plain Object ». Ces classes sont la représentation directe des tables et de leur colonne en base de données. 
Pour certains traitements ces objets contiendront trop de données et alourdiront l’application. C’est pourquoi des classes intermédiaires « Data Transfert Object » 
seront utilisées au sein de la couche métier de l’application. Ces classes intermédiaires sont des versions allégées des « Plain Object ».
